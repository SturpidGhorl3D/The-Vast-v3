# 🌌 THE VAST: Космическая Одиссея

**Высокопроизводительная процедурная 2D космическая симуляция нового поколения.**

THE VAST — это сложный симулятор выживания и экспансии в глубоком космосе. Каждая деталь вашего корабля — от формы корпуса до расположения реакторов — влияет на физику, тепловыделение и боевую эффективность. Мир управляется честной ECS-симуляцией (System-Component-Entity), обеспечивающей тысячи активных объектов в реальном времени.

---

## 🛠 Установка и Запуск

Игра подготовлена для быстрого старта на любой ОС:

1.  **Windows**: Запустите файл `setup.bat`. Он проверит наличие Node.js, установит все библиотеки и запустит игру автоматически.
2.  **Linux / macOS**: Используйте скрипт `setup.sh`. Сделайте его исполняемым (`chmod +x setup.sh`) и запустите. Он выполнит установку и запуск.
3.  **Ручной запуск**:
    ```bash
    npm install
    npm run dev
    ```
    Доступ по адресу `http://localhost:3000`.

## 🎮 Руководство Пилота

### 👫 Мультиплеер: Как играть с друзьями
Система построена на комнатах (Rooms):
*   **Хост**: В главном меню выберите «Многопользовательская» -> «Открыть сервер». Выберите ваш мир, установите пароль (опционально) и получите **Room ID**.
*   **Клиент**: Введите полученный **Room ID** и пароль в меню «Подключиться».
*   **Синхронизация**: Ваши корабли, снаряды и действия будут синхронизированы в общем секторе.

### 📐 Проектирование в Эдиторе
*   **Закон Симметрии**: Используйте оси отражения для баланса маневренности.
    *   *Вертикальная кнопка эдитора (Y)* отражает по горизонтали (X = -X).
    *   *Горизонтальная кнопка эдитора (X)* отражает по вертикали (Y = -Y).
*   **Масса и Энергия**: Тяжелые корабли требуют больше маршевых двигателей и мощных гироскопов (`GYRO`). Все модули потребляют энергию реакторов (`REACTOR`). Если энергии не хватает, системы будут отключаться.
*   **Геометрия**: Удаляя точки полигона, эдитор автоматически пересчитывает «скелет» корабля для корректной работы отсеков.

### ⛏ Добыча и Выживание
*   **Сканирование**: Правой кнопкой мыши (или долгим нажатием на мобильном) откройте контекстное меню цели, чтобы узнать состав астероида.
*   **Сбор ресурсов**: Снаряды шахтерских лазеров выбивают частицы материалов, которые автоматически собираются магнитной ловушкой при наличии свободного места в `CARGO` отсеках.

## 📱 Мобильное управление
Игра полностью оптимизирована для сенсорных экранов.
*   **Джойстик**: Появляется автоматически при касании экрана слева.
*   **Интерфейс**: Размер шрифтов и элементов управления адаптируется для комфортной игры на смартфонах (уменьшенный базис 14px).
*   **Контекст**: Долгий тап на объекте заменяет правый клик мыши для вызова тактического меню.
