# 🌌 THE VAST: Космическая Одиссея

**Высокопроизводительная процедурная 2D космическая симуляция.**

THE VAST — это амбициозный проект в жанре космической песочницы, сочетающий в себе процедурную генерацию галактики, детальное судостроение и глубокую экономическую логистику. Исследуйте бесконечные просторы, добывайте ресурсы, торгуйте и сражайтесь в мире, где каждая деталь вашего корабля имеет значение.

---

## 🛠 Установка и Запуск

Для запуска игры на локальной машине следуйте этой инструкции:

1.  **Клонируйте репозиторий** (или скачайте файлы).
2.  **Установите зависимости**:
    ```bash
    npm install
    ```
3.  **Настройте переменные окружения**:
    Создайте файл `.env` в корневой директории и добавьте ваш API ключ для работы игровых ИИ-систем:
    ```env
    GEMINI_API_KEY=ваш_ключ_здесь
    ```
4.  **Запустите сервер разработки**:
    ```bash
    npm run dev
    ```
5.  **Откройте игру**: Перейдите по адресу `http://localhost:3000` в вашем браузере.

## 🎮 Игровые Подсказки

### 👫 Мультиплеер: Как играть с другом
Игра поддерживает синхронизацию состояния через WebSockets.
*   **Создание сессии**: Просто запустите игру и поделитесь ссылкой (если используете облачный хостинг или проброс портов).
*   **Синхронизация**: Игроки в одном секторе будут видеть движения и действия друг друга в реальном времени.
*   **Важно**: Убедитесь, что ваш брандмауэр позволяет соединения по порту, на котором запущен сервер (по умолчанию 3000).

### 🛠 Проектирование корабля
*   **Масса и Энергия**: Чем больше корпус, тем выше его масса и потребление энергии. Не забывайте устанавливать достаточное количество реакторов.
*   **Симметрия**: Используйте кнопки зеркального отображения в редакторе (X/Y), чтобы создавать сбалансированные конструкции. Помните: вертикальная кнопка включает горизонтальную симметрию (относительно оси Y).
*   **Слои (Deck)**: Используйте функционал палуб для разделения жизненно важных узлов и бронирования.

### 💎 Добыча и Ресурсы
*   **Сканирование**: Используйте целевое окно (Targeting), чтобы определить содержание ресурсов в астероидах.
*   **Логистика**: Для переработки материалов требуются соответствующие модули (Fabricator/Refinery) и свободное место в трюме.

## 🚀 Технологический Стек
*   **Движок**: ECS на TypeScript + Babylon.js (WebGPU/WebGL).
*   **Интерфейс**: Next.js 15, React 19, Tailwind CSS.
*   **Сервер**: Node.js, Socket.io для мультиплеера.

*Разработано с заботой о производительности и глубине симуляции.*
